Mozfest and Matrix, Part 2

Tue Sep 12, 2017 by Ross Schulman in  MozFest 2017  MozFest 2017, matrix, portable, decentralization

My estimates of what we need to accomplish before MozFest are roughly: Get a RaspberryPi 3 and install Raspbian Lite as well as synapse Install PostgreSQL to serve as synapse’s database instead of sqlite Configure synapse Engage with MozFest wranglers to determine appropriate network settings (hostname, DNS, WiFi, etc.) Figure out a carrying case or some way to make it all portable Figure out a display of some kind Determine how to display visualizations, and what synapse APIs are usable for this purpose Prepopulate logins database table with registration info from MozFest (and ponder privacy concerns) Think about data retention post MozFest There’s almost certainly other tasks I’m not thinking of at the moment.
